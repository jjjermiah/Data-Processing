1    # Fundamentals of a data-processing pipeline
2    
3    Automation in the form of robust pipelines described as _code_ is a key foundation for success of large-scale data processing
4    as it demonstrates it is technically possible to process the data without intervention
5    
6    ## When to use a pipeline/workflow
7    - Anytime you find yourself running sequential commands that are predictable/repetitive (with mostly the same settings everytime)
8    - When operations can be run in parallel, especially when individual steps take a few minutes to run
9    
10   ## Re-use existing code, tools, containers, and other assets
11   - chances are someone else has already tackled a similar problem, and you may be able to reuse/augment their work
